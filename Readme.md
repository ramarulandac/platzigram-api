# Platzigram REST API

API build following TDD good practices to support platzigram functionalities. 
Using libraries like JSON webtoken, micro, uuid_base62, request, request-promise, AVA, standard, linter.

## Description

Three microservices build:

 * Pictures
 * Users
 * Authentication

## Installation

npm install

## Usage

 * micro -p port users.js
